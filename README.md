# Antigravity Release Mirror

自动同步 Google Antigravity Windows x64 安装包到 GitHub Release。

## 说明

本仓库通过 GitHub Actions 每天自动检查并同步 Google Antigravity 的最新版本到 Release 页面。

## 下载

访问 [Releases](../../releases) 页面下载最新版本的 Antigravity Windows x64 安装包。

## 自动化

- 每天 UTC 00:00 自动执行检查
- 如果发现新版本，自动下载并创建 Release
- 也可以在 Actions 页面手动触发工作流

## 官方链接

- 官方网站: https://antigravity.google
- 官方下载: https://antigravity.google/download
